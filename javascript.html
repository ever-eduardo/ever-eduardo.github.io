JavaScript

Javascript es un lenguaje de programacion usado como uno de los componentes fundamentales de la World Wide Web junto con HTML y CSS. JavaScript es un lenguaje de alto nivel, dinamicamente tipeado, soporta programacion orientada a objetos basada en prototipos, y soporta programacion funcional asi como programacion concurrente.

JavaScript es usado para darle funcionalidad a las paginas web en el lado del front-end, aunque en ultimamente se usa tambien como lenguaje para back-end. Javascript es un lenguaje bastante flexible que permite diversos paradigmas de programacion y es el lenguaje de base para acceder y manipular el DOM o Documento Object Model que es el documento que se crea en los navegadores web y que tiene estructura de arbol donde cada nodo de ese arbol es un elemento html. Con javascript se pueden escribir librerias y frameworks web para manipular el DOM y hacer de la web una experiencia fundamentalmente dinamica.

Cada navegador web tiene incorporado un interprete de Javascript, y se puede acceder a ese interprete a traves de la consola del navegador. Asi mismo, tambien existen run-time engines para correrlo fuera del ambito especifico de los navegadores como son por ejemplo NodeJS y Deno que permiten ampliar el espectro de uso de javascript a ambitos como el back-end e incluso ciertas bases de datos.

El estandar que define el lenguaje se llama ECMAScript y una lista con algunas de sus versiones es la siguiente:

  1. ECMA-262 la mas reciente especificacion en la version 13 2022
  2. ES6 en 2015 actualmente las mas usada
  3. ES2018 en 2018
