JavaScript

JavaScript es un lenguaje de programacion usado como uno de los componentes fundamentales de la World Wide Web junto con HTML y CSS. JavaScript es un lenguaje de alto nivel, dinamicamente tipeado, soporta programacion orientada a objetos basada en prototipos, y soporta programacion funcional asi como programacion concurrente.

JavaScript es usado para darle funcionalidad a las paginas web en el lado del front-end, aunque en ultimamente se usa tambien como lenguaje para back-end.

Cada navegador tiene incorporado un interprete de JavaScript, pero tambien existen run-time engines para correrlo fuera del ambigo de los navegadores como por ejemplo NodeJS y Deno
