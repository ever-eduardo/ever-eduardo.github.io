CSS 3

CSS 3 es un lenguaje de estilos en cascada que se usa para describir la presentacion de un documento escrito en un lenguaje de marcado como el HTML o XML. Este lenguaje fue creado para poder separar el contenido de la presentacion. Algunos ejemplos de aspectos de presentacion pueden ser el color de fondo, las fuentes, la alineacion, etc.
