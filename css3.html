CSS 3

CSS es un mecanismo de estilos en cascada que se usa para describir la presentacion de un documento escrito en un lenguaje de marcado como el HTML o XML. Este mecanismo fue creado para poder separar el contenido de la presentacion y poder asi especializar estos dos campos del desarrollo web. Algunos ejemplos de aspectos de presentacion pueden ser el color de fondo, las fuentes, la alineacion, asi como otros aspectos mas recientes como las animaciones, etc.

La version actual es CSS3 y ha podido adaptarse a un mundo digital lleno de dispositivos que tienen diferentes caracteristicas en sus pantallas. Es asi como se habla ahora de responsive web design y todo esto es posible gracias a la evolucion de CSS.

Una de las caracteristicas nuevas de CSS3 son los selectores, los efectos en textos, las estructuras multi colomna junto con las pseudo-clases y pseudo-elementos. Todo este conjunto permite realmente hacer que un disenador web o un ingeniero front end pueda inspirarse e inspirar a otros con su arte. La web ya no es solo para transmitir contenidos sino que tambien tiene un elemento estetico y artistico gracias a estos elementos CSS.
