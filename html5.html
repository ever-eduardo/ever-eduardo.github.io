HTML 5

HTML significa HyperText Markup Language y es uno de los pilares que da forma a la World Wide Web . Este lenguaje define el significado y la estructura del conteido web. La palabra Hypertext se refiere a links, vinculos o referencias que conectan unas paginas web con otras en un mismo sitio web o con otros. Los vinculos o links son el aspecto fundamental de la web. Ello permite tener una experiencia de lectura de contenidos y de procesamiento de informacion diferente a la lectura tradicional que es lineal y se haace de principio a fin. Con los hipervinculos se estructura la informacion de forma hiper textual con referencias y ya no es exclusivamente lineal sino que el espacio semantico puede tener una estructura semejante al grafo mas que a la linea.

HTML usa etiquetas o marcas para anotar texto, imagenes o cualquier otro contenido que se quiera mostrar en la web. Cada etiqueta o marca se dice que es un elemento HTML y algunos ejemplos de esos elementos son head, title, div, entre otros.

Un elemento HTML inicia con un signo '<', luego el nombre de la etiqueta y luego se cierra con el signo '>'. Es una buena practica escribir el nombre del elemento en letras minuculas, aunque es importante notar que ellas no son sensibles a esos cambios. En la imagen que viene a continuacion se puede ver la anatomia de un elemento HTML.

HTML ha tenido varias versiones. La primera version se la debemos al fisico Tim Bernes Lee del CERN. La idea era hacer un protocolo que permitiera compartir informacion entre los cientificos del mundo. Tim desarrolla el protocolo y el primer navegador y en ese moment comienzo el mundo de la World Wide Web.

En la actualidad HTML 5 es la quinta version de este lenguaje de marcado y se hizo publica en 2008. De todas formas HTML es un estandard que vive y por lo tanto evoluciona en cada momento como se puede leer en la pagina oficial de su especificacion. El sentido de las ultimas actualizaciones tienes que ver con la incorporacion de elementos multimedia, nuevos elementos sintacticos y semanticos que permiten estructurar los documentos HTML de forma que sea semanticamente significativa tanto para humanos como para motores de busqueda.
