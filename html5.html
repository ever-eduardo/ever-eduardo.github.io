HTML 5

HTML significa HyperText Markup Language y es uno de los pilares de la web. Este lenguaje define el significado y la estructura del conteido web. La palabra Hypertext se refiere a vinculos que conectan unas paginas web con otras en un mismo sitio web o con otros. Los vinculos o links son el aspecto fundamental de la web.

HTML usa etiquetas o marcas para anotar texto, imagenes o cualquier otro contenido que se quiera mostrar en la web. Cada etiqueta o marca se dice que es un elemento HTML y algunos ejemplos de esos elementos son head, title, div, entre otros.

Un elemento HTML inicia con un signo '<', luego el nombre de la etiqueta y luego se cierra con el signo '>'. Es una buena practica escribir el nombre del elemento en letras minuculas

HTML 5 es un lenguaje de marcado que se usa para estructurar y presentar contenido sobre la World Wide Web. La version 5 se hizo publica en 2008 y se hizo un update en 2014. El sentido de la ultima actualizacion tuvo que ver con la incorporacion de elementos multimedia, nuevos elementos sintacticos y semanticos.
